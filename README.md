# Infinite Canvas Blog

A personal blog built on an infinite canvas using [tldraw](https://tldraw.dev).

## Setup

```bash
npm install
cp .env.example .env
# Add your tldraw hobby key to .env
npm run dev
```

## Stack

- **Vite** + **React**
- **tldraw SDK** — infinite canvas
- Hosted on **GitHub Pages** (coming soon)
